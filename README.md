# Marketing Campaign Performance Analysis

## Objective
Analyze marketing campaign data to identify top-performing campaigns, 
high-value customer segments, and demographic spending patterns to 
enable data-driven targeting decisions.

## Tools Used
- Python (pandas) — Data cleaning and feature engineering
- MySQL — KPI queries and customer segmentation analysis
- Excel — Interactive dashboard with charts and slicers

## Dataset
- Source: Kaggle — Marketing Campaign Dataset
- Size: 2,216 customer records | 31 features
- Domain: Retail/E-commerce customer engagement data

## Key Findings
- Campaign 4 achieved the highest acceptance rate (164 responses)
- High Value customers (spend > $1000) respond 7x more to campaigns than Low Value segments
- PhD-educated customers have the highest average spend ($676.73)
- Young customers (< 35) spend the most on average ($810.57) despite being only 61 in number
- Overall campaign response rate: 30% across 2,216 customers
